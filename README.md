### 项目简介

使用 vue3 以及其对应的 element-ui (element-plus) 开发;
其内容是 将平时开发中看上去比较牛的功能 实现
目前 已将动画视频 实现 (技术栈: canvas)

### 重点功能点说明

#### 动态主题色

--theme-color 主题色
--theme-weaken-color-10 一般为 hover 的颜色

#### canvas 图片视频动画

通过 canvas 制作特效将图片弄成视频的形式播放
